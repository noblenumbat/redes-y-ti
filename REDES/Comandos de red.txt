
Comando de red 

Comando		        Sintaxis Ejemplo		    Qué hace
nslookup	        nslookup www.google.com		descubrir la dirección IP de un nombre de dominio
netstat		                			        ver conexiones activas
ipconfig /renew					                pedir una nueva dirección
tracert						                    saltos de red en Linux
ip addr show					                es como ipconfig /all
netstat -r -n					                ver mi gateway
netstat -rn					                    ver mi gateway
arp -d                                          borrar la tabla ARP de un host
arp -a                                          ver tabla de ARP en un host 
sudo resolvectl show-cache                      muestra entradas dns en cache
resolvectl status                               ver servidor dns configurado
ping -4 "URL"                                   traduce una dirección IPv6 a una direccion IPv4         
nslookup > set type=mx > cisco.com              identificar servidores de correo
